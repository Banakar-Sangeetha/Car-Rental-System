#  Car Rental Management System
*A Java Swing Application for Managing Cars, Customers & Rentals*

---

##  Overview
The **Car Rental Management System** is a desktop-based Java application designed to manage a small car rental service. It allows users to add cars, register customers, rent vehicles, and handle returns through a clean and intuitive **Java Swing GUI**.

This project demonstrates strong usage of **Object-Oriented Programming (OOP)**, event-driven GUI


##  Features

###  Vehicle Management
- Add new cars with ID, brand, and model
- View all registered cars
- Real-time availability check

###  Customer Management
- Register customers with unique ID
- View all customers

###  Rental Operations
- Rent a car to a customer
- Automatically updates car availability
- Return rented cars
- Tracks rental and return timestamps

###  Validations
- Duplicate ID detection (Cars/Customers)
- Input validation with error dialogs
- Clean status messages and confirmations


---

##  How It Works
1. Launch the application  
2. Menu options appear on the left  
3. Output logs appear on the right  
4. Perform actions like:
   - Adding cars/customers  
   - Renting cars  
   - Returning cars  
   - Viewing reports  

Sample data is auto-loaded for testing.
